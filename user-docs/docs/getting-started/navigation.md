# Navigate the app

The sidebar organizes current work, workspace configuration, and personal settings. What you see can vary by deployment and role.

## Sidebar

- **Home:** The chat composer and recent work.
- **Agent Ops:** Runs that you can access.
- **Observability:** Operational views available in the workspace.
- **Projects:** A project tree and a link to all projects.
- **Shared:** Projects and runs that other people shared with you.
- **Resources:** Skills, MCP servers, runtime profiles, MCP policies, guardrails, modes, and roles.
- **Users:** An **Admin** section visible only to administrators. It opens user management.
- **Settings:** Personal appearance, credentials, SOUL, role-model preferences, Git identity, and desktop-only connection or update settings.

The older **Sources** navigation is not part of the current sidebar. Project and integration capabilities can still be available when your deployment configures them.

## Command palette

Press <span className="kbd">⌘</span> <span className="kbd">K</span> or <span className="kbd">Ctrl</span> <span className="kbd">K</span> to open the command palette.

Use it to search available destinations, recent items, and settings. Its choices depend on the current app build, workspace configuration, and permissions.
