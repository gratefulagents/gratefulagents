# Shared with me

**Shared** lists projects and runs that other workspace users shared with you.

## Open Shared

Select **Shared** in the sidebar. You can also search for it in the command palette when the destination is available.

## Find a shared resource

The page groups results into **Runs** and **Projects**. Each result shows its name, status, person who shared it, and your permission. Use the search field to filter the visible results.

Select a result to open it. If no results appear, no resource has been shared with you or the search does not match.

## Permissions

The owner chooses **Viewer** or **Collaborator** when sharing. The app displays your assigned permission and hides or disables actions you cannot use.

A Viewer can inspect the shared resource to the extent the app permits. A Collaborator can have additional actions, such as working from a shared project, but available actions depend on resource type, state, workspace policy, and deployment.

Ask the owner to change or revoke access if you need a different permission.
