# Users

**Users** is an administrator-only page. It appears in the sidebar's **Admin** section only for users with the `admin` role.

## Manage users

The page lists registered users with their name, username, email, role, and last login. Use search to narrow the list.

Administrators can:

- Select **Promote to admin** for a non-admin user.
- Change another user's role among `admin`, `member`, and `viewer`.
- Delete another user's account.

You cannot change your own role or delete your own account from this page.

Deleting an account permanently removes it and signs the user out everywhere. Their runs and projects are not deleted. Confirm organizational policy before changing roles or deleting an account.
