## Default Permission

Allows starting the in-app Google OAuth flow

#### This default permission set includes the following:

- `allow-start-google-oauth`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`google-oauth:allow-start-google-oauth`

</td>
<td>

Enables the start_google_oauth command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`google-oauth:deny-start-google-oauth`

</td>
<td>

Denies the start_google_oauth command without any pre-configured scope.

</td>
</tr>
</table>
